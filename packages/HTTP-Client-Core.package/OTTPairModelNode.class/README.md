An OTTPairModelNode represents an item of the OTTPairModel.

Instance Variables
	parent:		<OTTPairModel>
	key:		<String>
	value:		<String>
	balloonText:		<String>

parent
	- parent OTTPairModel

key
	- key of the represented pair

value
	- value of the represented pair

balloonText
	- balloon text which should be shown when the user hovers the item
